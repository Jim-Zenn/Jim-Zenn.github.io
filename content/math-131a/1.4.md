{{% definition name="Upper bound" status="" %}}
$S \subseteq F  $ an ordered field . An element $X $ s.t. X \geq S all $s \in S $ is upper bound for $S $.
{{% /definition %}}

{{% definition name="Lower bound" status="" %}}
$S \subseteq F  $ an ordered field . An element $X $ s.t. X \leq S all $s \in S $ is lower bound for $S $.
{{% /definition %}}




$X$ is upper bound for $S $. If $x \in S $, then $x$ is maximum of $S$.

$X$ is lower bound for $S $. If $x \in S $, then $x$ is minimum of $S$.

Closed interval: $[3, 7] $, $3 \leq x \leq 7$.

Half-open interval: $[3, 7), 3 \leq x < 7 $.

Open interval: $(3, 7), 3 < x < 7$.

{{% definition name="least upper bound (supremum)" status="" %}}

If $S $ has an upper bound $x $ which is $\leq $ all the upper bounds, then X is **least upper bound**.

{{% /definition %}}

{{% definition name="greatest lower bound (infrimum)" status="" %}}

If $S $ has an lower bound $x $ which is $\geq$ all the lower bounds, then X is **greatest lower bound**.

{{% /definition %}}

{{% note name="" %}}

By definition,

**Maxima** are suprema.

**Minima** are infima.

{{% /note %}}

{{% note name="" %}}

$\N $ and $\Q $ have the same size, but $\R $ is bigger.

{{% /note %}}


{{% note name="" %}}

$inf(\emptyset) = + \infty$
$sup(\emptyset) = - \infty$

if $ S \neq \emptyset + x $  sup S

if $ S \neq \emptyset + y $  inf S

Then $y \leq x$

Let $ s \in S $

$ y \leq s \leq x $


{{% /note %}}

{{% axioms name="Completeness Axiom" %}}

If $ S \subseteq \R $ and $ S $ is bounded above (i.e. it has some upper bound). Then S has supremum.

{{% /axioms %}}

{{% example name="" %}}

$(1, \sqrt{ 2 }) = s$

inf s = $1 \in Q$

sup S not $\sqrt{ 2 } \not\in Q$

{{% /example %}}

{{% note name="" %}}

Showing a number isn't algebraic isn't easy.

e.g. How to show $\pi, e $ are not algebraic?

{{% /note %}}

{{% note name="" %}}

Complete ordered field $\R$.

Then every set bounded below has an infimum.

{{% /note %}}
