{{% definition name="Linear combination" %}} 
$V$ is a vector space over field $F$ and $S$ a non-empty subset of V. A vector $v \in V$ is called a **linear combination** of vectors from $S$ if there exists **FINITE** many vectors $v_1, v_2, ..., v_n \in S$ and $a_1, a_2, ..., a_n \in F$ s.t. $$v = a_1v_1 + a_2v_2 + ... + a_nv_n$$
Furthermore, in this case, we say that $w$ is a linear combination of $v_1, v_2, ..., v_n$ with coefficients $a_1, a_2, ..., a_n$.
{{% /definition %}}

{{% definition name="Span" %}} 
$V$ is a vector space over field $F$, $S$ is a non-empty subset of $V$.
The span of $S$ or the linear span of $S$ denoted by $Span(S)$ is the set of <u>all possible</u> linear combination of vectors of s.

We define $span(\emptyset) = \set{0}$

<small>$\abs{S}$ could be finite or infinite.</small>
{{% /definition %}}

{{% theorem name="" index="1.5" %}} 

$V$ is a vector space over field $F$, if $S \subset V$, then $Span(S) \leq V$.

Moreover, if $W \leq V, S\subset W$, then $Span(S) \leq W$.

{{% proof name="" %}} 
**case 1**:
if $S$ is $\emptyset$, $y$ definition $Span(s) = {0} \leq V$.

**case 2**:
$S$ is non-empty $(S \neq \emptyset)$
$\exists V \in S$

(W1).$0 = 0\cdot v \in Span(S)$

(W2).$x, y \in Span(S)$

$x \in Span(S) \implies \exists v_1, v_2, ..., v_m \in S$ and $a_1, a_2, ..., a_n \in F$ s.t.
$x = a_1v_1 + a_2v_2 + ... + a_nv_n$


$y \in Span(S) \implies \exists w_1, w_2, ..., w_n \in S$ and $b_1, b_2, ..., b_n \in F$ s.t.
$y = b_1w_1 + b_2w_2 + ... + b_nw_n$

$x + y = a_1v_1 + a_2v_2 + ... + a_mv_m + b_1w_1 + b_2w_2 + ... + b_nw_n$

$\set{v_1, v_2, ...,v_n, w_1, w_2, ..., w_n}$ at most $m+n$ elements

$x+ y \in Span(S)$

(W3).$x\in Span(S)$ and $\lambda \in F$
$\lambda x \in Span(S)$
$x = a_1v_1 + a_2v_2 + ... + a_nv_n$
$\lambda x = \lambda a_1v_1 + \lambda a_2v_2 + ... + \lambda a_nv_n$
$\lambda x \in Span(S)$

Q.E.D.

{{% /proof %}}

{{% /theorem %}}

{{% theorem name="" index="" %}} 
Let $S$ be a subset of a vector space $V$ over Field $F$. Then $Span(S)$ is a subspace of $V$. Moreover, $Span(S)$ is the smallest subspace containing $S$.

{{% proof name="" %}} 
$S \subseteq Span(S)$

$v \in S, v = 1\cdot v \in Span(S)$

How to proof Span $S$ is the smallest set?

We need to proof that \forany $W \leq V$ containing $S$ 

i.e. $S \subseteq W$, need to proof that $Span(S) \leq W$

<br>

$x\in Span(S)$

$\therefore \exists x_1, x_2,..., x_n \in S$ and $a_1, a_2, ..., a_n \in F$ s.t. $x = a_1x_2 + a_2x_2 + ... + a_nx_n$

$S\subseteq W$
$\therefore x_i \in S \implies x_i \in W$

$x_i \in S \implies x_i \in W \implies a_ix_i \in W$

i.e. $a_1x_1 + a_2x_2 + ... + a_nx_n \in W$

$\implies x\in W$

$\therefore span(S) \leq W$

{{% /proof %}}
{{% /theorem %}}


{{% definition name="" %}}

In this case we say $S$ **generates** $V$.

{{% definition name="linearly dependent" %}} 
Let $V$ be a vector space over field $F$.

A subset $S \subseteq V$  is called **linearly dependent** if $\exists$ finite many vectors $v_1, v_2, ..., v_n \in S$ and scalar $a_1, a_2,..., a_n$ NOT ALL ZERO s.t. $$a_1v_1 + a_2v_2 + ... + a_nv_n = 0$$

{{% /definition %}}

{{% example name="$\R^2$" %}} 

$S= \set{(1,0), (0,1), (2,-3)}$

$-2(1,0) + 3(0,1) +1(2,-3) = (0,0)$

{{% /example %}}

{{% example name="" %}} 
$T = {(1, 0), (0,1), (5,0)}$

$-5(1,0) + 0(0,1) + 1(5,0) = (0,0)$

{{% /example %}}

{{% definition name="linearly independent" %}} 
Let $V$ be a vector space over $F$ a subset of $S\subseteq V$ is called **linearly independent** if it is NOT linearly dependent.

Equivalently, if for **every finite** subset of vector $v_1, v_2, ..., v_n \in S$ the only solution of the equation (in $a_i$) $$a_1v_1 + a_2v_2 + ... + a_nv_n = 0$$
is 
$$a_1=a_2=...=a_n=0$$

{{% /definition %}}


{{% example name="$\R^2$" %}} 
$S=\set{(1,0),(0,1)}$ is linearly independent.

{{% proof name="" %}} 
$a(1,0) + b(0,1) = (0,0)$

$\implies (a,b) = (0,0)$

$a= 0, b=0$

$S$ is linearly independent.
{{% /proof %}}
{{% /example %}}

{{% example name="$\R^2$" %}} 
$S=\set{(1,1),(3,-3)}$ is linearly independent.

{{% proof name="" %}} 
$a(1,1) + b(2,-3) = (0,0)$

$(a+2b,a-3b) = (0,0)$

$$
\begin{align\*}
a+2b=0 \\\\\
a-3b=0
\end{align*}
$$

$\implies (a,b) = (0,0)$

$a= 0, b=0$

$T$ is linearly independent.
{{% /proof %}}
{{% /example %}}

{{% theorem name="" index="" %}} 
$V/F$, let $S$ be a linearly independent subset of $V$, and $v$ a vector in $V$.
Then $S \cup \set{v}$ linearly dependent $\iff$ $v \in Span(S)$
{{% proof name="" %}} 
Assume $v\in Span(S)$ :

$\therefore \exists v_1, v_2, ..., v_n \in S$ and $a_1, a_2, ..., a_n \in F$ s.t. $$v=a_1v_1 + a_2v_2 + ... + a_nv_n$$

$a_1v_1+a_2v_2 + ... a_nv_n + (-1)v = 0$

$S\cup \set{v}$ is linearly dependent.

<br>

Assume $S\cup \set{v}$ is linearly dependent:

$\therefore v_1, v_2, ..., v_n, v \in S\cup \set{v}$ 
and $a_1, a_2, ..., a_n \in F$ s.t.

$$a_1v_1+a_2v_2 + ... + a_nv_n + av = 0$$
and at least one of $a_1, a_2, ... ,a_n, a$ is non-zero.

Claim $a \neq 0$ 

If  a = 0, then all have 
a_1v_1 + a_2v_2 + ... + a_nv_n = 0 

since $S$ is linearly independent $\set{v_1, v_2, ..., v_n} \subseteq S$ is also linearly independent.

from theorem we get $a_1=a_2=a_3= ... = a_n = 0= a$

this is a contradiction to the condition $a_1, a_2, ..., a_n, a$ not all zero

$\therefore a \neq 0$

$a_1v_1+a_2v_2 + ... + a_nv_n + av = 0$
$av = -a_1v_1 -a_2v_2 - ... -a_nv_n$
$v = \frac{-a_1}{a}v_1 \frac{-a_2}{a}v_2 + ... +\frac{-a_n}{a}a_nv_n$
$\implies a \in Span(S)$

{{% /proof %}}
{{% /theorem %}}
