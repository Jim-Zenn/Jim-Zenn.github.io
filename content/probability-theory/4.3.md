{{% definition name="" status="" %}}

Let $X $ and $Y $ be two random variable. 

The covariance of $X $ and $Y $ is defined by

$\cov(X, Y) = E\brac{(X - E\brac{ X }) (Y - E\brac{ Y })} = E\brac{ XY } - E\brac{ X } E\brac{ Y }$

The correlation coefficient of $X $ and $Y $ such that $\var {X}, \var {Y} \neq 0 $ is $\rho (X, Y) = \frac{ \cov{X, Y}}{ \sqrt[  ]{ \var{X} \var{Y}}} $

{{% /definition %}}

{{% properties name="" %}}

1. $\cov{X, Y} = \cov {Y, X} $
2. $\cov{X, X} = \var{X} $
3. For constants $a, b \in \R$, $\cov (X, aY + b) = a \cov(X, Y)$
4. $\cov{X, Y+Z} = \cov{X, Y} + \cov{X, Z} $

{{% /properties %}}

{{% example name="" %}}

$X, Y ~ Uniform [0, 1] $, $X, Y$  are independent, $f\_{X\_Y} (t) = ? $

$F\_{X\_Y}(t) = f\_X * f\_Y(t) = \int\_{ -\infty }^{ +\infty } f\_X(x) f\_Y(t - x) \d x $

$0 \leq x \leq 11, 0 \leq t - x \leq 1, t-1 \leq x \leq t $

$f\_X(x) f\_Y(t- x) = \begin{cases}
1 &, \text{if }\max{ 0, t- 1 } \leq x \leq \min{ 1, t } \br
0&, \otherwise \br
\end{cases}$
$F\_{X+Y}(t) = \int\_{ x \in \R } f\_X(x)f \_Y(t - x) \d x = \begin{cases}
\min{ 1 ,t } - \max{ 0, t-1 }&, t \in (0, 2) \br
0&, \otherwise
\end{cases} = \begin{cases}
t &, t \in (0, 1) \br
2 - t, t \in (1, 2) \br
0, \otherwise
\end{cases}$
